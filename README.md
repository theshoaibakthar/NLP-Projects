# Natural Language Processing Projects

## Overview
This repository showcases a collection of Natural Language Processing (NLP) projects, demonstrating practical implementation of various text processing techniques, machine learning models, and deep learning approaches. Each project highlights different aspects of NLP, from basic text preprocessing to advanced sentiment analysis.

## Projects

### 1. [SMS Spam Classification](/SMS-Spam-Classification)
A comprehensive spam detection system implementing multiple text classification approaches:
- Bag of Words (BOW) with Multinomial Naive Bayes
- TF-IDF Vectorization with Multinomial Naive Bayes
- Word2Vec with Random Forest Classifier
- **Skills demonstrated**: Text preprocessing, Feature engineering, Model development, Classification metrics

### 2. [Kindle Review Sentiment Analysis](/Kindle-Review-Sentiment-Analysis)
Sentiment analysis on Amazon Kindle reviews using various NLP techniques:
- Implemented multiple text vectorization methods (BOW, TF-IDF, Word2Vec)
- Built and compared different classification models
- Processed and analyzed the Kindle Reviews dataset
- **Skills demonstrated**: Large-scale data processing, Sentiment Analysis, Word embeddings, Model evaluation
